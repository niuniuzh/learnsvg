# 坐标定位

SVG坐标系统是：以页面的左上角为(0,0)坐标点，坐标以像素为单位，x轴正方向是向右，y轴正方向是向下。

<svg version="1.1"
     baseProfile="full"
     width="300"
     height="300"
     xmlns="http://www.w3.org/2000/svg">
     <rect x="10" y="30" width="100" height="100" fill="green" />
</svg>

定义一个矩形，即从左上角(10,30)开始，向右延展100px，向下延展100px，填充绿色的正方形。
